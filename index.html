Linux command prompt

Arietta has a SSH client installed so you need an SSH client to get the access to the system console.

  • If you're on Google Chrome, install the plug-in Secure Shell. When installed open it from Chrome Apps menu.
  • On Linux you could use ssh by typing, from a terminal session: ssh acme@arietta.local
  • On Windows you could use putty.exe
In any case the login data are:

  • Login as root: root Password: acmesystems
